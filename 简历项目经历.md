# 简历项目经历 - 健身助手

---

## 健身助手 App

**技术栈**：uni-app 3.0 + Vue 3.4 + Vite 5.2 + uCharts 2.5

**项目描述**：
- 跨平台健身管理应用，支持Android、iOS、微信小程序、H5等10+平台
- 包含102个动作库、智能训练建议、成就系统、数据可视化分析等核心功能
- 采用纯本地架构，无需服务器，数据安全可靠

**技术亮点**：
1. 使用Canvas API实现高性能数据可视化，支持训练容量趋势图、力量进步曲线等
2. 设计智能训练建议算法，基于历史数据推荐训练重量和次数
3. 实现游戏化成就系统，包含成就解锁逻辑和实时进度追踪
4. 采用模块化架构，封装4个核心工具模块，代码复用性高
5. 使用深色主题设计，提升用户体验

**项目成果**：
- 1天完成MVP版本开发
- 打包发布10+个APK版本
- 实现13个页面，覆盖完整健身流程
