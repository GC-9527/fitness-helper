# HBuilderX 运行指南

> 使用 HBuilderX 运行项目到 Android 基座

---

## 📋 前置条件

1. **安装 HBuilderX** - 已安装（版本 4.87）
2. **安装 Android 模拟器或连接真机**
3. **项目代码已完成**

---

## 🚀 运行步骤

### 1. 在 HBuilderX 中打开项目

1. 打开 HBuilderX
2. 点击菜单栏 **文件** → **打开目录**
3. 选择项目文件夹：`D:\weixinxcx\uni-app-version`
4. 点击 **选择文件夹**

### 2. 配置 manifest.json

1. 在项目管理器中找到 `manifest.json`
2. 双击打开配置页面
3. 确认以下配置：
   - **应用名称**: 训练日常
   - **应用版本**: 5.0.0
   - **AppID**: __UNI__1D6311F

### 3. 运行到 Android 基座

#### 方式一：使用 Android 模拟器

1. 启动 Android 模拟器（如夜神模拟器、雷电模拟器）
2. 在 HBuilderX 中点击菜单栏 **运行** → **运行到手机或模拟器** → **运行到 Android App 基座**
3. 等待编译完成
4. 应用会自动安装并启动在模拟器上

#### 方式二：使用真机

1. 使用 USB 线连接 Android 手机
2. 开启手机的 **开发者选项** 和 **USB 调试**
3. 在 HBuilderX 中点击菜单栏 **运行** → **运行到手机或模拟器** → **运行到 Android App 基座**
4. 等待编译完成
5. 应用会自动安装并启动在手机上

## 🐛 常见问题

### 问题 1：提示"未检测到基座"

**解决方案**:
1. 确保模拟器已启动或手机已连接
2. 检查 USB 调试是否开启
3. 尝试重新插拔 USB 线
4. 在 HBuilderX 中点击 **工具** → **ADB** → **重置 ADB**

### 问题 2：编译失败，提示符号链接错误

**解决方案**:
1. 以管理员身份运行 HBuilderX
2. 或者右键 HBuilderX 图标 → **以管理员身份运行**

### 问题 3：应用安装成功但闪退

**解决方案**:
1. 检查 manifest.json 配置是否正确
2. 查看 HBuilderX 控制台日志
3. 确保所有文件编码为 UTF-8

### 问题 4：无法识别设备

**解决方案**:
1. 安装手机对应的 USB 驱动
2. 在 HBuilderX 中点击 **运行** → **运行到手机或模拟器** → **刷新设备列表**
3. 尝试更换 USB 接口

3. **调试** - 可以在 HBuilderX 控制台查看日志输出
4. **性能** - 开发模式性能不如发行模式，属正常现象
