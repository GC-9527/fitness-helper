# 记组记重功能实现完成

## 功能概述
实现了类似训记的记组记重功能，支持自重和自由重量两种训练类型。

## 主要修改

### 1. 训练页面 (pages/index/index.vue)
- **新增训练类型选择器**：每个动作可以选择"自重"或"自由重量"
- **组数控制**：右上角显示 +/- 按钮调整组数
- **组详情列表**：
  - 显示每组的重量和次数输入框
  - 自重动作只显示次数
  - 自由重量显示重量(kg)和次数
  - 已完成的组显示绿色勾选标记
  - 当前待完成的组高亮显示
  - 点击数值可直接修改
- **数据同步**：调整组数时会自动添加/删除组详情

### 2. 动作详情页 (pages/exercise-detail/exercise-detail.vue)
- **底部操作栏**：添加快速添加到训练功能
- **快速配置**：
  - 可调节组数、次数
  - 自由重量时可输入重量
  - 可切换训练类型（自重/自由重量）
- **一键添加**：添加到训练后自动跳转到首页

### 3. 计划页面 (pages/plans/plans.vue)
- 更新所有创建/使用计划的方法，确保动作包含组详情数据
- 选择动作时初始化组详情

## 数据结构
```javascript
{
  id: '动作ID',
  name: '动作名称',
  part: '肌群部位',
  sets: 4,              // 总组数
  reps: 12,             // 默认次数
  weight: 0,            // 默认重量
  exerciseType: 'freeweight', // 'bodyweight' | 'freeweight'
  completedSets: 0,     // 已完成组数
  folded: true,         // 是否折叠
  setDetails: [         // 每组详情
    { weight: 20, reps: 12 },
    { weight: 20, reps: 12 },
    { weight: 20, reps: 10 },
    { weight: 20, reps: 8 }
  ]
}
```

## 使用流程
1. 在动作库选择动作 → 进入动作详情页
2. 在底部设置组数、次数、重量（自由重量）
3. 选择训练类型（自重/自由重量）
4. 点击"添加到训练"
5. 在训练页面展开动作卡片
6. 可修改每组的重量和次数
7. 点击"完成第X组"记录完成

## 界面特点
- 组详情区域清晰显示每组数据
- 已完成的组有绿色边框和勾选标记
- 当前待完成的组高亮显示
- 输入框禁用状态区分已完成/未完成
- 锁定状态下无法修改数据
