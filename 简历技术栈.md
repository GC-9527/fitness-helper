# 健身助手项目 - 技术栈

## 📋 项目简介
健身助手是一款跨平台健身管理应用，采用前后端分离架构，前端基于uni-app框架开发，支持Android、iOS、微信小程序、H5等多端发布。项目包含102个动作库、智能训练建议、成就系统、数据可视化分析等核心功能。

---

## 🛠️ 技术栈

### 前端框架
- **uni-app 3.0** - 跨平台应用开发框架
- **Vue 3.4** - 渐进式JavaScript框架
- **Vite 5.2** - 下一代前端构建工具

### 核心技术
- **Composition API** - Vue 3组合式API
- **响应式数据管理** - Vue 3响应式系统
- **本地存储** - uni-app Storage API
- **Canvas图表** - 自定义图表绘制

### UI组件库
- **uni-app Components** - 官方组件库
- **自定义组件** - 封装定制化组件（custom-navbar等）

### 第三方库
- **uCharts 2.5** - 跨端图表库（@qiun/ucharts）
  - 折线图、趋势图等数据可视化

### 开发工具
- **HBuilderX** - uni-app官方IDE
- **Trae SOLO** - AI辅助开发工具

---

## 🎯 核心功能实现

### 1. 数据可视化模块
- 使用Canvas API实现自定义图表绘制
- 训练容量趋势图、力量进步曲线
- 支持数据筛选和动态更新

### 2. 智能建议系统
- 基于历史训练数据的算法推荐
- 渐进式超负荷训练建议
- 实时更新训练参数

### 3. 成就系统
- 成就解锁逻辑设计
- 实时进度追踪
- 分类展示和筛选

### 4. 本地数据持久化
- 训练记录、成就数据本地存储
- 数据格式设计和优化
- 兼容性处理

## 💡 技术亮点

1. **零成本开发** - 无需服务器，纯本地应用
2. **一次开发，多端运行** - 一套代码支持10+平台
3. **高性能数据可视化** - Canvas原生绘制，流畅不卡顿
4. **模块化架构** - 功能模块独立封装，易于维护
5. **深色主题设计** - 护眼模式，提升用户体验
